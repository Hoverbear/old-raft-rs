Proposes an entry to be appended to the replicated log. This will only return once the entry has been durably committed. Returns Error when the entire cluster has an unknown leader. Try proposing again later.

Queries an entry from the state machine. This is non-mutating and doesn't go through the durable log. Like .propose() this will only communicate with the leader of the cluster.
